# 超级大乐透预测辅助系统

基于 FastAPI + Vue 3 + SQLite 的超级大乐透（DLT）预测辅助项目。从 500.com 抓取真实历史开奖数据，结合本地 LLM（Ollama / OpenAI 兼容接口）生成预测号码，并提供号码形态分析、回测模拟、用户投注跟踪等辅助功能。

> ⚠️ **免责声明**：彩票每期独立随机，任何分析方法都无法提高中奖概率。本项目仅用于技术研究与数据分析学习，不构成任何投注建议。请理性购彩，量力而行。

## 核心功能

### 🎯 智能预测
- 接入本地 Ollama（如 glm-4.7:cloud）或任意 OpenAI 兼容 LLM 生成号码
- 每注预测附带选号理由与分析说明
- 支持一次生成 1/3/5 注，多注可批量采纳
- 注入结构化统计数据（遗漏值/和值/奇偶比/大小比/跨度/上期形态）辅助 LLM 决策

### 📊 号码分析
提供 11 个分析子模块，帮助建立结构化分析思维：

| 模块 | 功能 |
|---|---|
| 组合诊断 | 输入号码→输出形态体检报告（和值/奇偶比/大小比/跨度/区间/连号/重号/AC值/同尾号）+ 综合评分 |
| 遗漏值 | 35 个前区 + 12 个后区号码的遗漏期数排行 |
| 和值分布 | 历史和值分桶统计 + 最近 30 期折线趋势 |
| 奇偶大小 | 奇偶比 / 大小比历史分布 |
| 奖池趋势 | 奖池金额历史走势 + 8 亿阈值线 |
| 频率统计 | 前/后区号码出现频率 + 遗漏值高亮 |
| 期望值 | 一等奖理论期望值计算 + 奖池历史分位 + 正/负期望判断 |
| 策略对比 | 热号/冷号/随机/均衡 4 种策略历史命中率横向对比 |
| 历史复盘 | 隐藏开奖→你选号→揭晓→命中对比 + 综合评分 |
| 策略沙盒 | 自定义规则（和值/奇偶比/大小比/跨度/连号）→批量生成→历史回测 |
| 实时预警 | SSE 实时推送遗漏≥15 期号码 + 奖池状态 |

### 🔮 回测模拟
- **AI 预测回测**：选期号→AI 基于该期前数据预测→对比实际开奖算奖金（支持多注）
- **固定号码长期模拟**：输入一组号码→模拟在所有历史期购买→统计累计收益
- **AI 预测+长期模拟**：AI 生成号码→从指定期一直买到最新期（支持多注）

### 💰 奖金计算
- **双规则支持**：26014 期前用旧 9 级规则，26014 期起用新 7 级规则（按期号自动切换）
- **8 亿奖池上浮**：新规则下奖池≥8 亿时固定奖金上浮（如三等奖 5000→6666 元）
- **真实奖池数据**：抓取每期实际奖池金额、一/二等奖中奖注数和单注奖金

### 🎫 用户投注跟踪
- 采纳 AI 预测或手动输入号码→记录到数据库
- 自动跟踪每注开销（2 元/注）
- 开奖后自动评估命中数、中奖等级、奖金金额
- 汇总卡片：总花费/总中奖/净收益/中奖率

### ⏰ 自动化
- 定时抓取：周一/三/六 21:40 自动抓取最新开奖数据
- 自动评估：开奖后自动结算所有待评估投注
- 数据持久化：SQLite 存储，重启不丢失

## 技术栈

| 层 | 技术 |
|---|---|
| 后端 | FastAPI + SQLAlchemy ORM + SQLite + APScheduler/Timer |
| 前端 | Vue 3 (Composition API) + Vite + Tailwind CSS |
| LLM | Ollama 本地模型 / OpenAI 兼容 API |
| 数据源 | 500.com（requests + BeautifulSoup 解析 HTML 表格） |
| 实时通信 | SSE (Server-Sent Events) |

## 快速开始

### 环境要求
- Python 3.10+
- Node.js 18+
- （可选）Ollama 本地服务 + glm-4.7:cloud 模型

### 后端启动

```bash
cd backend
python -m venv ../.venv
../.venv/Scripts/activate          # Windows
# source ../.venv/bin/activate     # Linux/Mac
pip install -r requirements.txt
python -m uvicorn main:app --host 127.0.0.1 --port 8888 --reload
```

首次启动会自动从 500.com 抓取最近 200 期历史数据。

### 前端启动

```bash
cd frontend
npm install
npm run dev
```

浏览器访问 http://localhost:5175

### LLM 配置

在「智能预测」生成号码前，需配置 LLM。支持两种方式：

1. **Ollama 本地**：安装 Ollama → 拉取模型 → 后端自动检测
2. **OpenAI 兼容 API**：通过环境变量或应用内配置 API Key 和 Base URL

未配置 LLM 时，系统会自动降级为启发式随机选号。

### Docker 部署

```bash
docker-compose up -d
```

## 数据来源

- **开奖数据**：500.com（`https://datachart.500.com/dlt/`）
- **抓取字段**：期号、开奖日期、前区 5 球、后区 2 球、销售额、奖池滚存金额、一等奖中奖注数/单注奖金、二等奖中奖注数/单注奖金
- **抓取时机**：首次启动自动抓取；之后每周一/三/六 21:40 自动增量更新
